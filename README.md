# Laboratory Work #10: University System

## Overview
This project implements an Object-Oriented Programming (OOP) solution for a University System. The system manages various entities related to a university including employees, faculties, institutes, projects, and research activities.

## Project Structure

```
lab-10-university-system/
├── src/                    # Source code (Java files)
│   ├── Employee.java       # Base employee class
│   ├── University.java     # Main university class
│   ├── Project.java        # Project management class
│   ├── ResearchAssociate.java  # Research associate implementation
│   ├── Faculty.java        # Faculty class
│   ├── Institute.java      # Institute management
│   └── Lecturer.java       # Lecturer class
├── diagrams/              # Visual Paradigm diagrams and models
├── screenshots/           # Project execution screenshots
└── README.md              # This file
```

## Features

- **Employee Management**: Handle different types of employees in the university
- **Department Structure**: Organize faculties and institutes
- **Project Management**: Track research and academic projects
- **Research Activities**: Manage research associates and their activities
- **Object-Oriented Design**: Implements key OOP principles including inheritance, polymorphism, and encapsulation

## Technologies Used

- **Language**: Java
- **IDE**: IntelliJ IDEA
- **Modeling Tool**: Visual Paradigm

## Contents

- **src/**: Contains all Java source code files
- **diagrams/**: Contains UML diagrams and models created with Visual Paradigm
- **screenshots/**: Contains execution output and results screenshots
